如何用Github学生包嫖到海外的云端服务器

我平常科学上网用的是Github学生包里的Digital Ocean，送一年200刀抵扣券，如果是只用于科学上网的话可以申请一个月4刀的最低套餐，配置单核，512MB运存，一个月500GB流量。当然也可以开更高规格的服务器，毕竟这200刀只有一年的有效期，怎么分配看你自己喽。

其实有一说一，一个月4刀给500GB流量还很稳定其实自费的话也还可以。

学生包申请页面：GitHub Student Developer Pack - GitHub Education

其实Github学生包里包含了两个项目可以弄到云服务器，一个是Azure，另一个是Digital Ocean

在此对两个介绍一下：

  1. Azure的特点就是稳！在有些关键时期都不会出问题，而且全球的节点很多，但也有个特点就是贵。开局给你100刀，如何使用就看你自己了（亲测霓虹节点比HK节点延迟还要低就离谱）

  2. Digital Ocean的特点就是量大管饱，但是延迟都普遍在200ms左右，它的节点不多，不过我推荐旧金山的节点，因为别的节点响应很慢，别看这印度和新加坡虽然离咱近，但是延迟和美国的鸡房却差不多。

  3. 二者均不会限速，速度上限取决于你的运营商和你的地理位置，有些运营商把外网访问卡得很死，比如长城宽带和广电之类的

如何申请到GitHub学生认证？

我个人是提交了三次学生认证才通过的，提交学生证或者用edu邮箱均可。

当收到这封邮件时就代表学生认证通过了

当然如果实在嫌麻烦直接闲鱼上买也可以，买前问清楚学生包里的优惠是不是都是未激活的状态，毕竟闲鱼奸商蛮多的。

Digital Ocean开服流程（推荐）

GitHub Student Developer Pack - GitHub Education

先注册好Digital Ocean账号，这个过程可能需要梯子，不然访问速度会很慢很慢。

注册Digital Ocean我当时注册是需要5刀的激活费用，需要你的Paypal账户，这个其实也算好办，Paypal是支持连接国内银联卡的，直接绑定一下就好

再来到这里将自己的Github账号和Digital Ocean进行连接

连接成功后就会带到这个页面

在Digital Ocean左侧的Billing中可看到自己的余额

接下来新建项目，随意就好

新建一个Droplets，这个Droplets就是云服务器的意思，起的名字很有特色，啊

在此之后就是喜闻乐见的选配置环节了

想要在服务器上配置离线下载或者Docker之类杂七杂八服务的话就整个大一点的存储空间，后期不够其实也可以加扩展盘，这一点都是很自由的。如果只是梯子的话最低配置就足够，反正只是个流量转发和数据包加密，又吃不了多少性能。

选完配置等服务器初始化结束后会给一个公网ip，用这个公网ip和选配置时自己设置的密码登录服务器SSH

在此我提供个脚本

下载到云端后直接bash执行，按照流程走就可以，用酸酸乳的客户端就行
